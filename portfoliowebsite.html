Java Program - Battleship Game

In this Java program, I used different coding techniques to create my own version of the classic "Battleship" board game. In this photo, you can see how the program functions. When part of a ship is hit, the box turns red. When the entire ship has sunk, a GUI pop up message will inform the user. If the user selects a spot where there is no ship, the box will turn white so the user knows they've already tried that spot. The placement of the ships are randomly generated per game.
